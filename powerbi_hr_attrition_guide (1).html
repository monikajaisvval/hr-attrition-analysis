Part 1 — Install & load data
1

Download Power BI Desktop (free)

Go to microsoft.com/power-bi and download Power BI Desktop. It's completely free and runs on Windows.

2

Load the CSV file

Open Power BI Desktop → click "Get data" → choose "Text/CSV" → select your hr_attrition_clean.csv file → click Load.

Use the cleaned CSV from your Python notebook, not the raw Kaggle file — it already has the Attrition_num column (0/1) which makes calculations easier.
Part 2 — Create calculated columns & measures
3

Create an attrition rate measure (DAX)

In the right panel, click "New measure" and type this DAX formula:

Attrition Rate = DIVIDE( COUNTROWS(FILTER('hr_attrition_clean', 'hr_attrition_clean'[Attrition] = "Yes")), COUNTROWS('hr_attrition_clean'), 0 )
This is your most important measure — it will power all 4 KPI cards at the top.
4

Create an age group column

Go to Table view → click "New column" and enter:

Age Group = SWITCH(TRUE(), 'hr_attrition_clean'[Age] <= 25, "18-25", 'hr_attrition_clean'[Age] <= 35, "26-35", 'hr_attrition_clean'[Age] <= 45, "36-45", "46-60" )
Part 3 — Build the visuals one by one
5

Add 4 KPI cards (top row)

Click "Card" visual 4 times and configure each:

Total Employees → COUNT(EmployeeNumber)
Employees Left → COUNTROWS where Attrition = Yes
Attrition Rate % → your DAX measure above
Avg Age → AVERAGE(Age)
6

Attrition by department — clustered bar chart

Insert → Clustered bar chart. Set:

Y-axis: Department X-axis: Attrition Rate (your DAX measure) Legend: Attrition (Yes/No) Colors: Yes = coral/red, No = teal
7

Overtime vs attrition — donut chart

Insert → Donut chart. Set:

Legend: OverTime Values: Attrition Rate measure
This will instantly show the 30.5% vs 10.4% split that is your strongest insight.
8

Attrition by age group — bar chart

Insert → Clustered bar chart. Set:

Y-axis: Age Group (column you created in step 4) X-axis: Attrition Rate measure Sort: Sort by attrition rate descending
9

Income vs attrition — scatter chart

Insert → Scatter chart. Set:

X-axis: MonthlyIncome Y-axis: Attrition Rate measure Legend: Attrition Size: COUNT(EmployeeNumber)
If the scatter looks cluttered, switch to a box-and-whisker or use a binned bar chart instead — both are easier to read in Power BI.
Part 4 — Add slicers & finishing touches
10

Add slicers (interactive filters)

Insert → Slicer. Add these 3 slicers so the dashboard becomes interactive:

Department
OverTime (Yes/No)
Gender
Slicers are what impress recruiters most — clicking "Sales" should update all 4 visuals instantly. This shows you understand interactive reporting.
11

Style & theme the dashboard

View → Themes → choose "Executive" or "Frontier" for a clean professional look. Then:

- Set report title: "HR Employee Attrition Analysis" - Use consistent font: Segoe UI - Attrition = Yes color: #C0392B (red) - Attrition = No color: #1D9E75 (teal/green) - Add a text box with your name + date
12

Export & share

Take a screenshot for Naukri (Windows: Win + Shift + S). To share interactively, publish to Power BI Service (free account needed) and copy the share link.
